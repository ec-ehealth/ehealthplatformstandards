StructureDefinition: BePractitionerRole

Belgian federal profile for a practitioner role. Initially based on the functional description of the NIHDI.

The official URL for this profile is: 

https://www.ehealth.fgov.be/standards/fhir/StructureDefinition/be-practitionerrole
